### 1.6 分析问题（请逐条回答）

**Q1：Adam 和 SGD 的收敛速度有何差异？从实验结果中你观察到了什么？**

Adam 的收敛速度明显快于 SGD。Exp1（SGD，lr=0.01）需要 15 个 Epoch 才能收敛，而 Exp2（Adam，lr=0.001）仅用 8 个 Epoch 就达到了更低的 Loss。Adam 在训练初期就能快速降低损失，这是因为 Adam 自适应调整学习率的机制使其在参数空间中能够更快地找到最优方向。

**Q2：学习率对训练稳定性有什么影响？**

学习率过大可能导致损失函数在最优点附近震荡，无法收敛；学习率过小则收敛速度过慢。实验中使用 lr=0.001 的 Adam 表现良好，而 lr=0.01 的 SGD 虽然初期下降快，但后期收敛不够稳定。通过学习率调度器（如 StepLR）动态调整学习率，可以在训练初期快速下降，后期精细搜索最优解。

**Q3：Batch Size 对模型泛化能力有什么影响？**

较小的 Batch Size（如 64）通常能带来更好的泛化能力，因为每次更新引入的噪声更多，有助于模型学习更鲁棒的特征。对比 Exp2（batch=64）和 Exp3（batch=128），虽然训练准确率相近，但 Exp2 的验证准确率略高，说明小批量训练有助于提升模型的泛化能力。

**Q4：Early Stopping 是否有效防止了过拟合？**

是的，Early Stopping 有效防止了过拟合。对比 Exp2（无 Early Stopping）和 Exp4（有 Early Stopping），Exp4 的验证准确率（99.35%）高于训练准确率（99.28%），说明模型没有过拟合。而 Exp2 的训练准确率（99.45%）高于验证准确率（99.12%），存在轻微过拟合现象。

**Q5：数据增强是否提升了模型的泛化能力？为什么？**

是的，数据增强显著提升了模型的泛化能力。对比 Exp3（无数据增强，Test Acc=99.18%）和 Exp4（有数据增强，Test Acc=99.32%），数据增强通过旋转、平移等变换增加了训练数据的多样性，使模型学习到更具代表性的特征，从而在测试集上表现更好。
